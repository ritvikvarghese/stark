# log

the raw layer. every working day gets an entry even if it's three lines:
what i built, what broke, what i learned. commit daily so the graph tells
the story. never clean up history, never delete a failed approach, move it
here and say why it failed.

one file per day: `day-00.md`, `day-01.md`, ...

## template

```markdown
# day N — YYYY-MM-DD

**goal:** [the day's checkpoint from course/]

## what i built
-

## what broke
-

## what i learned
-

## photos / clips
-
```
